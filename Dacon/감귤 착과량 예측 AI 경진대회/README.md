# 🏆 감귤 착과량 예측 AI 경진대회

## 📊 대회 정보
- **주최**: DACON
- **평가 지표**: RMSE

## 🎯 프로젝트 개요
### 1. 목표
- 감귤 나무의 착과량을 정확하게 예측
- 농업 데이터를 활용한 회귀 모델 개발

### 2. 데이터
- 감귤 나무 생육 데이터
- 기상 데이터
- 토양 데이터
- 과거 착과량 데이터

### 3. 접근 방법
1. **데이터 전처리**
   - 결측치 처리
   - 이상치 탐지
   - 데이터 정규화

2. **Feature Engineering**
   - 생육 특성 추출
   - 기상 데이터 가공
   - 토양 특성 생성

3. **모델링**
   - NGBoost 모델 활용
   - Oversampling 기법 적용
   - 앙상블 기법 적용

## 📚 프로젝트 구조
```
감귤 착과량 예측 AI 경진대회/

```

## 🏆 주요 성과
- NGBoost 모델을 활용한 예측
- Oversampling을 통한 데이터 불균형 해결
