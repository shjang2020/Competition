# 🏆 UOS 빅데이터 알고리즘 경진대회

## 📊 대회 정보
- **주최**: DACON
- **평가 지표**: RMSE

## 🎯 프로젝트 개요
### 1. 목표
- 시계열 데이터를 활용한 예측 모델 개발
- 다양한 딥러닝 모델의 성능 비교 및 최적화

### 2. 데이터
- 시계열 데이터
- 시계열 특성
- 외부 요인 데이터

### 3. 접근 방법
1. **데이터 전처리**
   - 시계열 데이터 정규화
   - 이상치 처리
   - 결측치 보간

2. **Feature Engineering**
   - 시계열 특성 추출
   - HP Filter 적용
   - 외부 요인 반영

3. **모델링**
   - LSTM 모델 활용
   - Neural Prophet 적용
   - DARTS 기법 활용
