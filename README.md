## 🎯 참가 목표
1. 다양한 도메인의 문제 해결 경험
2. 최신 AI/데이터 사이언스 기술 학습
3. 실무 적용 가능한 프로젝트 경험
4. 네트워킹 및 커뮤니티 참여

   - 특성 엔지니어링

2. 머신러닝/딥러닝
   - 회귀/분류 모델
